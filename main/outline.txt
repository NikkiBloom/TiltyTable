Main class
    - looping code
    - attaches other classes together
    - communicates to motors (may want to call a motor class or library)

HMI class
    - read inputs from knobs
    - output to display

Math class
    - inverse kinematics
    - motor PID functions (maybe, depends on motor libraries)
